# Projeto: Busca Professores

Este projeto da aula de Engenharia de Software tem como objetivo simular uma classe em Java responsável por pupular uma página web com informações sobre os horários de atendimento dos professores do Inatel. As informações teoricamente seriam retornadas de um servidor remoto em formato JSON.

## Testes unitários

Usando Mock, foram realizados testes unitários, mais especificamente 10 testes para casos de sucesso e 10 testes para caso de falha. Utilizando a biblioteca Junit.
